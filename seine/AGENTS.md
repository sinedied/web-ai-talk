# Seine

A modern online demo store, similar to another famous one named after a river. Seine is built with raw HTML, CSS, and JavaScript, without any frameworks for demo simplicity. 

## Workflow

Whenever you make a new change to the website:
1. Record a short video (few seconds) in .mp4 of the feature in action using Playwright, under temp/.
2. Run dev server, and send the URL to me on telegram so I can test your changes on my phone, along with the video.
3. Never close issue until I say it's good to go
